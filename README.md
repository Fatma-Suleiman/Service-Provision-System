# Project Jirani

A dynamic application that bridges the gap between local service providers and customers. Built with a high-trust interface where users can discover, evaluate, and book services instantly, reducing the friction found in traditional service procurement.

## Features

- Discover local services easily
- Book services instantly
- Evaluate and review providers
- Trust-focused interface

## Tech Stack

- React
- Node.js
- REST APIs
- Service Booking
